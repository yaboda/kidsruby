Sphero Commands

The following list of commands can be used with your Sphero:

Sphero.start(port) do...end

Connects to Sphero and starts to execute the commands between 'do' and 'end'

port - how to connect to a particular Sphero

roll speed, heading

Makes Sphero start to roll in a particular direction

speed - a number between 0 and 255

heading - a number from 0 to 359 for which direction to go in

stop

Makes Sphero stop

heading= h

Makes Sphero change direction when already rolling

h - a number from 0 to 359 for which direction to go in

rgb r, g, b

Changes Sphero LED color

r - a number between 0 and 255

g - a number between 0 and 255

b - a number between 0 and 255

keep_going duration

Tells Sphero to keep doing what it is already doing for a certain amount of time

duration - a number for how many seconds to continue

ping

Pings Sphero to see if it is responding

FORWARD

A constant value, same as 0

BACKWARD

A constant value, same as 180

RIGHT

A constant value, same as 90

LEFT

A constant value, same as 270

There are more commands, but these are the most important. For more information, take a look at the Sphero gem documentation on Github.
